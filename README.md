# Coffee Shop SQL Analysis

Proyek ini berisi analisis data transaksi coffee shop menggunakan SQL untuk memahami pola penjualan dan performa bisnis.

## Gambaran Dataset
Dataset berisi lebih dari 100.000 baris data transaksi yang mencakup informasi:
- ID transaksi
- Tanggal dan waktu transaksi
- Jumlah pembelian per transaksi
- Informasi toko (ID dan lokasi)
- Informasi produk (ID, kategori, tipe, dan detail produk)
- Harga satuan produk

## Analisis yang Dilakukan
Analisis dilakukan menggunakan query SQL, meliputi:
- Pemeriksaan awal data (jumlah data, nilai kosong, dan rentang nilai)
- Identifikasi produk dengan penjualan terbanyak
- Perhitungan total pendapatan harian
- Analisis kategori produk dengan pendapatan tertinggi
- Pola penjualan berdasarkan jam transaksi
- Tren penjualan berdasarkan hari dalam seminggu
- Analisis lanjutan menggunakan GROUP BY dan HAVING

## Tools yang Digunakan
- MySQL
- DBeaver
- GitHub
- Microsoft Word

## Output
- coffee_shop_analysis.sql
  Berisi kesimpulan query SQL yang digunakan untuk data understanding, analisis penjualan, dan analisis pengelompokan data.
- coffee_shop_analysis.pdf
  Berisi laporan analisis yang mencakup temuan utama dari data, pola penjualan yang terlihat, rekomendasi bisnis sederhana, serta kesimpulan berdasarkan hasil analisis.

## Catatan 
Dataset mentah (CSV) tidak disertakan dalam repository ini.
Analisis dilakukan menggunakan dataset coffee shop yang disediakan sebagai bagian dari studi kasus Data Catalyst Bootcamp.
